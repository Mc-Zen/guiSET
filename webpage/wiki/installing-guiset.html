    Installing guiSET

    Installing guiSET is simple - just install it like every other Processing library. (We didn't register it yet, please install manually at the moment) In the Processing IDE select Sketch/Import Library/Add Library. This opens the Contribution Manager where you can search for and install guiSET .

    Or you can download the source code here from github extract the .zip file and put the guiSET folder into the libraries folder of your Processing location (usually in "documents" in your computer).
