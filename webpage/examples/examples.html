    • Examples

    Available examples (with links to github repository):

    • Hello_World - First step with guiSET.
    • Most_Components - Demonstration of (almost) all available GUI components. Also shows special constructors and setters.
    • Border_Layout - Usage of anchors and resizable windows. More about that in article about dynamic resizing and anchors here.
    • Border_Layout2 - Usage of anchors and resizable windows (variation).
    • Arduino_Serial_Monitor - Implementation of a serial monitor to connect with a usb device. Can be used to send and receive messages.
    • Custom_Component - Demonstration on creating a new (custom) component class. More about that in article about creating own components here.
    • Notepad - Implementation of a simple text editor with guiSET including opening and saving any text file format. Usage of menus and resizing.
